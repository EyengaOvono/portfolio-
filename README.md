## Deploy on GitHub Pages

1. Put these files at the **root** of a repository on your own GitHub account
   (a dedicated repo such as `portfolio` keeps the URL clean).
2. In the repository: **Settings → Pages**.
3. Under **Build and deployment → Source**, choose **Deploy from a branch**.
4. Select the branch that holds these files and the **`/ (root)`** folder, then **Save**.
5. After 1–2 minutes the live URL appears at the top of the Pages settings
   (e.g. `https://<username>.github.io/portfolio/`).

No build command is needed — GitHub Pages serves the static files directly.
